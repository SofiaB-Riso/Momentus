# 📖 Sobre o Projeto

O **Momentus** é uma plataforma desenvolvida para auxiliar pessoas sem experiência em organização de eventos.

O sistema permite criar e gerenciar pequenos eventos, como:

* 🎂 Aniversários
* 🎉 Confraternizações
* 🥳 Festas em geral
* 👨‍👩‍👧‍👦 Reuniões familiares
* 📅 Outros eventos de pequeno porte

Nosso objetivo é tornar o planejamento mais organizado, reduzindo a preocupação dos usuários e centralizando todas as informações em um único lugar.

---

# 🚀 Funcionalidades

* 👤 Cadastro e autenticação de usuários
* 📅 Criação e gerenciamento de eventos
* 💰 Cálculo automático do orçamento
* 🔔 Notificações para os usuários
* 📩 Confirmação de presença dos convidados (RSVP)
* 📍 Definição do local do evento
* 🤝 Divisão de itens e responsabilidades entre os convidados
